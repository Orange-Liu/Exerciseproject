# Exerciseproject
根据视频做的练习项目，仅供学习使用

结构：
   e3-parent: 管理jar包版本，打包方式pom
   |-- e3-common: 存放公共工具
   |-- e3-manager: 管理dao、pojo、interface、service、web
      |-- e3-manager-dao : 数据处理层
      |-- e3-manager-pojo : 实体模块
      |-- e3-manager-interface : service接口模块
      |-- e3-manager-service : service接口实现类模块
      |-- e3-manager-web : 表现层模块
